### 1. Superpowers

- 类型：paradigm
- 证据等级：community-verified
- 更新状态：新收录
- 适用阶段：requirements / design / implementation / review / testing
- 关键来源：[GitHub 仓库](https://github.com/obra/superpowers)；[v6.1.0 Release](https://github.com/obra/superpowers/releases/tag/v6.1.0)；[Claude 官方 Plugin](https://claude.com/plugins/superpowers)

**证据要点**：采集时间 2026-07-02T14:41+08:00。GitHub API 显示 243,701 stars、21,623 forks、297 open issues，MIT license，archived=false；latest release v6.1.0 发布于 2026-06-30，commit 说明含 Codex marketplace install；README 列出 Claude 官方 plugin marketplace、OpenAI Codex plugin marketplace 与 Cursor `/add-plugin superpowers` 安装路径。

**项目介绍**：Superpowers 是一套面向 coding agent 的完整软件工程方法论：先通过对话提炼 spec、再产出可审计的实现计划，最后以 subagent-driven-development 分任务执行与审查。技能可自动触发，覆盖 Claude Code、Codex、Cursor 等多 harness。

**竞品/替代项**：

- [GitHub Spec Kit](https://github.com/github/spec-kit)：更偏官方 spec-driven CLI 与集成生态。
- [Addy Osmani Agent Skills](https://github.com/addyosmani/agent-skills)：更强调 lifecycle slash commands 与质量门禁。

**推荐理由**：社区热度极高且 release 节奏清晰；v6.1.0 明确支持 Codex marketplace，并进入 Claude 官方 plugin marketplace，是观察 subagent 工作范式与跨 harness 技能分发的关键样本。

**最佳使用范式**：用于 requirements、design、implementation、review、testing 全周期：安装后让 agent 先完成 spec 对齐与计划评审，再启用子代理分片实现；适合作为团队 agent SDLC 骨架对照。
