# overview 看板留白根治 — 设计决策与验收契约

> 架构师产出的**上下文材料 + 产出契约**。诊断与验收标准为准；**实现路径由实施窗口自主选择**，本文的"可选参考路径"仅供参考，可整体替换。

## 1. 背景

情报站首屏「今日摘要」看板中，**简讯卡**出现异常留白：折叠态右下角恒空一格；展开后该空格依旧悬在按钮上方（用户实测截图，简讯 5 条 / 7 条两种情况均复现）。用户要求：站在看板算法底层逻辑上优化，确保**自适应占位、不留异常空白**，并泛化审查看板算法是否值得进一步优化。

## 2. 根因诊断（现状，供理解，非实现约束）

- 看板 bullet 列表统一用常量 `OVERVIEW_BULLET_LIMIT = 3` 截断折叠。
- **仅简讯卡是两列网格**（`.overview-briefs .overview-bullets` 在 ≥740px 下 `repeat(2, ...)`），类目卡是单列。
- 「2 列 × 默认 3 条（奇数）」不整除 → 第二行只有 1 格有内容，**右下角恒空一格**。这就是留白根因；类目卡为单列，故无此问题——**该缺陷是简讯卡独有**。
- 展开逻辑把"其余条目"渲染进**另一个独立网格**（`overview-extra`），无法回填第一个网格的空格 → 展开态空格依旧存在。**双网格结构导致"补一处漏一处"**。

## 3. 泛化审查发现（本轮一并处理）

1. **奇偶/列数错配**：折叠数量与列数不整除是通病；即便把默认改成偶数，奇数"其余条数"在 extra 网格里仍会凑出新尾格——根源是**双网格 + 写死数量**的结构。
2. **魔法数字耦合**：同一个 `OVERVIEW_BULLET_LIMIT` 同时服务"单列类目卡"和"多列简讯卡"，一个常量无法对两种布局都最优，需解耦。
3. **连带缺陷（会被本次修改暴露）**：收起文案写死为"收起，回到前 **3** 条"。一旦简讯基准数量变化，文案即错误。disclosure 必须"知道自己的基准数量"，不得写死。
4. **等高拉伸留白（本轮不做，仅记录）**：`.overview-grid` 默认 `align-items: stretch` 会把同行矮卡拉高、底部留空——属"整体感"问题，本轮范围**不含**（如需再开轮次）。

## 4. 验收契约（硬性，必须全部满足）

1. **无内部空洞**：简讯看板在 **折叠态与展开态 × 桌面(2 列)/移动(1 列) × 任意条数（重点覆盖 3/4/5/7 及奇偶边界）** 下，条目之间、按钮上方均**不得出现异常空白格**。网格最末尾因奇偶不整除产生的、位于**最末位置**的单个尾格属正常，应尽量由展开/收起控件或布局策略消化，不得表现为悬空的洞。
2. **折叠数量自适应**：折叠态默认展示数量需与"当前有效列数"对齐（不再硬编码与列数错配的奇数）；桌面/移动列数不同也应各自成立。
3. **解耦魔法数字**：单列类目卡与多列简讯卡的折叠基准需可各自取值，不再由同一写死常量强绑。
4. **修复连带缺陷**：收起/展开文案必须反映**实际基准数量**，杜绝"回到前 3 条"这类写死。
5. **可逆 + 无障碍不回退**：保留现有可逆展开/收起（`aria-expanded`/`aria-controls`/键盘 Enter·Space/收起时焦点管理/`prefers-reduced-motion` 兜底），本次不得回退这些能力。
6. **零功能回归**：不改数据契约；不影响 gist 消费、`#/catalog` 资产总表、OG/favicon、类目卡跳转、筛选、路由等既有功能。类目卡当前无留白，改动不得引入新问题。
7. **改动范围**：仅 `site/app.js` + `site/styles.css`。

## 5. 可选参考路径（非约束，可自行替换）

- 折叠默认数量对齐列数（如桌面 2 列取偶数）——治标，但单独用不足以覆盖展开态奇数尾格。
- 单连续网格 + 逐条 `hidden` 切换，取代"基准网格 + 独立 extra 网格"，使展开时条目连续排布、尾格只可能出现在最末。
- 让"展开/收起"控件作为网格项**占据尾格**，主动消化不整除产生的空位。
- 以上可组合或另择方案，只要满足第 4 节验收契约。

## 6. 验证清单（完成后须自测并回报）

- `node --check site/app.js` 通过。
- 本地静态服务器（仓库根起 `http.server`，访问 `/site/`）桌面 + 移动视口目视：
  - 简讯 **3 / 4 / 5 / 7** 条，折叠态与展开态均无异常空白（尤其原右下角空格）。
  - 展开/收起可逆；文案数量正确（无"前 3 条"写死）。
  - 键盘 Enter/Space 可操作，收起后焦点回到按钮。
  - `prefers-reduced-motion` 下无异常动画。
  - 类目卡跳转、筛选、路由、`#/catalog`、gist 概览、OG/favicon 均无回归。
- 控制台无报错。
